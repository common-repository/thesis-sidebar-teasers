This plugin allows you to insert Thesis Teasers inside a sidebar. It's currently very simple and you'll need to update the styles to make it look good.

== Description ==

What does it do?
This plugin allows you to insert Thesis Teasers inside a sidebar. It doesn’t include all the options that you would normally expect. In a future release, it might.

Requirements:
This plugin for Wordpress requires that you’re using the Thesis theme (1.5 or higher.)

Questions & Comments?
Leave a comment at http://www.webdesigncompany.net/thesis-sidebar-teasers/ and I’ll respond as soon as I can.

== Installation ==

1. Upload plugin to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Insert the widget into your sidebar
